ğŸ“‹ Make it a Template Repository

Template repositories allow others (and yourself) to create new repos with the same structure instantly.

How to Enable Template

  1. Go to your repository settings 
    https://github.com/crazyief/claude-bus-framework/settings
  2. Scroll down to find "Template repository"

    It's in the General settings section, near the top.

  3. Check the box âœ… "Template repository"

    The setting saves automatically.

How to Verify It's a Template

  1. Visit your repository page 
    https://github.com/crazyief/claude-bus-framework
  2. Look for the green "Use this template" button

    If you see this button (instead of just "Code"), it's a template!

  3. Or check via CLI 
    gh repo view crazyief/claude-bus-framework --json isTemplate
# Output: {"isTemplate":true}

ğŸ’» Use from Another Computer

Starting fresh on a new machine? Here's how to get the framework running:

Step 1: Install Prerequisites

# Install GitHub CLI
sudo apt update && sudo apt install -y gh

# Install Claude Code CLI
npm install -g @anthropic-ai/claude-code

# Authenticate with GitHub
gh auth login

Step 2: Create Project from Template

# Create new project from template (recommended)
gh repo create my-awesome-project \
    --template crazyief/claude-bus-framework \
    --private \
    --clone

# Enter the project
cd my-awesome-project
âœ… What This Does
  • Creates a new private repo called my-awesome-project
  • Copies all files from the template
  • Clones it to your local machine
  • You own the new repo completely (not a fork)

Step 3: Start Claude Code

# Start Claude - framework auto-loads!
claude

# Run interactive setup
/setup

# Or start your first stage directly
/new-stage 1 "My First Feature"

Alternative: Clone Without Template

If template isn't enabled, you can still clone directly:

# Clone the framework
git clone https://github.com/crazyief/claude-bus-framework.git my-project
cd my-project

# Remove original git history and start fresh
rm -rf .git
git init
git add .
git commit -m "Initial commit from Claude Bus Framework"

# Start Claude
claude

ğŸ“ Use in Local Folder (Same Computer)

Want to start a new project on the same computer where you already have the framework?

Option A: Use Template via GitHub (Recommended)

# Create new project from your template
gh repo create another-project \
    --template crazyief/claude-bus-framework \
    --private \
    --clone

cd another-project
claude

Option B: Copy Folder Locally

# Copy framework to new location
cp -r /path/to/claude-bus-framework /path/to/my-new-project

# Go to new project
cd /path/to/my-new-project

# Reset git (start fresh history)
rm -rf .git
git init
git add .
git commit -m "Initial commit"

# Start Claude
claude

Option C: Use degit (No Git History)

# Install degit if needed
npm install -g degit

# Copy without git history
npx degit crazyief/claude-bus-framework my-new-project

cd my-new-project
git init
claude
âš ï¸ Important

Always start with a fresh git history for new projects. Don't keep the framework's commit history in your actual project.

Gate Protocol (Binary Voting)

ğŸ—³ï¸ Gate Rules
  • Binary Only: PASS or BLOCK (no "conditional")
  • Unanimous: All 6 agents must PASS
  • 2 Loops Minimum: Even if Loop 1 passes, Loop 2 required
  • 3 Loops Maximum: Escalate to user after 3 failures
